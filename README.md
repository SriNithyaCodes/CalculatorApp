## ✨ Key Features
- **Modern UI**: Sleek dark-mode aesthetic with vibrant button accents.
- **Glassmorphism Design**: Elegant card-based layout with subtle shadows and gradients.
- **Responsive Layout**: Fully optimized for mobile, tablet, and desktop screens.
- **Full Functionality**: Supports addition, subtraction, multiplication, and division.
- **Advanced Controls**:
  - `AC`: Clear the entire expression.
  - `DE`: Backspace function to delete the last character.

## 🛠️ Built With
- **HTML5**: Structured markup for accessibility and search engine optimization.
- **CSS3**: Advanced styling including Flexbox, Grid, and custom animations.
- **JavaScript**: Core logic handled efficiently for real-time calculations.
